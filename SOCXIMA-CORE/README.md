# SOCXIMA CORE

🚀 **Plataforma P2P Descentralizada sobre Blockchain Solana**

## Características

- 🎨 Interfaz Cuántica
- 📊 Panel Principal
- 🛡️ Panel Admin
- 🤖 Inteligencia Artificial
- 💱 Mercado P2P

## Tecnologías

- React 18
- Vite
- Styled Components
- Lucide React

```bash
npm install
```
