Integridad y firma:

  • SHA256SUMS
  • SHA256SUMS.asc (firma GPG)
  • Clave pública (GPG)

Verifica los archivos con sha256sum -c SHA256SUMS y gpg --verify SHA256SUMS.asc.

Instalación rápida en Termux

Si ejecutas Termux en Android, copia y pega este comando en tu terminal (recomendado usar la release oficial):

curl -fsSL https://github.com/eveliollo/SOCXIMA/releases/download/v0.0.0/install-socxima-termux.sh | sh

Alternativa manual (más segura):

  1. Descargar: wget https://github.com/eveliollo/SOCXIMA/releases/download/v0.0.0/socxima-termux.tar.gz
  2. Verificar SHA256: sha256sum socxima-termux.tar.gz
  3. Extraer: tar -xzf socxima-termux.tar.gz
  4. Instalar: cd socxima && ./install.sh

Sustituye las URLs de ejemplo por las rutas reales de tus assets o Release de GitHub si son diferentes.
